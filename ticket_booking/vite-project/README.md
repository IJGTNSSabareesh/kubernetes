# Travel Booking System

This is a React + Vite project for a Travel Booking System. It allows users to browse events/trips, select seats, authenticate, make bookings, and view a QR-coded ticket.

## Scripts
- dev: Start development server
- build: Build for production
- preview: Preview the production build

## Tech Stack
- React 19 + Vite 7
- React Router DOM 7
- Axios for API calls
- qrcode.react for QR code generation

## Development
1. npm install
2. npm run dev

## Backend Proxy
Proxied endpoints are configured in `vite.config.js`.
